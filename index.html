Genesis Framework

Wordpress Frameworks extend core WordPress functionalities by exposing and then enhancing specific functionalities beyond their native purpose to create a specific type of theme usually for specific types of user. This type of development requires a parent and child theme relationship. We would develop the parent theme using the framework for functionality and create "skin" child themes with CSS styles for design/modification.

HTML5 Blank Wordpress Boilerplate Theme

Webs most popular front-end framework. HTML5 Boilerplate helps you build fast, robust, and adaptable web apps or sites. As the fifth revision of the HTML standard, HTML5 aims to improve the language by providing support for the latest multimedia while remaining a user-friendly language. Some of the new features found in HTML5 include:

• New syntactic features, including video, audio and canvas elements • Integrated scalable vector graphics content • MathML for mathematical formulas

These features make it easier to include multimedia and graphical content without needing to use plugins and APIs. Meanwhile, HTML5 WordPress templates aim to take full advantage of the new functionality of the updated version of the HTML language.
